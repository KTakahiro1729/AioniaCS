# AioniaCS

慈悲なきアイオニアのための非公式キャラクターシート作成ツールです

## 使い方

最新版は [GitHub Pages](https://ktakahiro1729.github.io/AioniaCS/) で公開しています。

## ローカル下書き保存

ブラウザの `localStorage` に最新のキャラクターシート（キー: `aionia-character`）を自動保存するようになりました。ブラウザを閉じても下書きは復元されます。共有閲覧モード中は上書きされず、ドライブからの読み込みや新規作成時にはこの下書きがクリアされるため、常に意図した最新データが保持されます。
